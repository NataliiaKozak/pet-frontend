# Pet Shop Frontend

React course project: a small online shop for pet products.  
The application has several pages, an order form and interactive elements.  
The design follows the layouts created in Figma.

The frontend is built with React (Vite) and Redux Toolkit for state management.  
It uses React Router for navigation, Axios for HTTP requests and CSS Modules for styling.  
The app communicates with a Node.js / Express backend with a SQLite database.

## Live Demo

- Frontend (Vercel): https://pet-frontend-kappa.vercel.app/
- Backend API (Render): https://pet-backend-huwd.onrender.com

## Main Features

- View categories of pet products
- View all products and sales
- Add products to the cart / order form
- Send an order via the form

## Tech Stack

- **Frontend:**

  - React, Vite
  - Redux Toolkit (slices, store)
  - React Router DOM
  - Axios
  - React hooks
  - CSS Modules

- **Backend:**
  - Node.js, Express
  - Sequelize, SQLite
  - Deployment: Render

# Pet Shop Frontend

Курсовой проект по React: небольшой интернет-магазин товаров для домашних животных.
Приложение состоит из нескольких страниц, формы оформления заказа и интерактивных элементов.
Дизайн повторяет макеты, разработанные в Figma.

Фронтенд реализован на React (Vite) с использованием Redux Toolkit для управления состоянием.
Для навигации используется React Router, для HTTP-запросов — Axios, для стилизации — CSS-модули.
Приложение взаимодействует с backend-сервером на Node.js / Express с базой данных SQLite.

## Демонстрация

- Frontend (Vercel): https://pet-frontend-kappa.vercel.app/
- Backend API (Render): https://pet-backend-huwd.onrender.com

## Основной функционал

- просмотр категорий товаров для животных;
- просмотр всех товаров и акций;
- добавление товаров в корзину / форму заказа;
- отправка заказа через форму.

## Стек технологий

- **Фронтенд:**
  - React, Vite
  - Redux Toolkit (slices, store)
  - React Router DOM
  - Axios
  - React Hooks
  - CSS Modules

- **Бэкенд:**
  - Node.js, Express
  - Sequelize, SQLite
  - Деплой: Render
