# Pet Shop Frontend

A React course project: a small online shop for pet products. The application includes several pages, an order form, and interactive elements. The design follows layouts created in Figma.

The frontend is built with React (Vite) and Redux Toolkit for state management. It uses React Router for navigation, Axios for HTTP requests, and CSS Modules for styling. The app communicates with a Node.js / Express backend with a SQLite database.

## Live Demo

- Frontend (Vercel): https://pet-frontend-kappa.vercel.app/
- Backend API (Render): https://pet-backend-huwd.onrender.com

## Features

- Browse pet product categories
- View all products and current sales
- Add products to cart / order form
- Submit an order via the form

## Tech Stack

**Frontend:**
- React, Vite
- Redux Toolkit (slices, store)
- React Router DOM
- Axios
- React Hooks
- CSS Modules

**Backend:**
- Node.js, Express
- Sequelize, SQLite
- Deployed on Render
